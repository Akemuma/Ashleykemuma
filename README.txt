# 💼 Ashley Kemuma's Portfolio

Welcome to my personal portfolio! 🚀 This website showcases my projects, skills, and experiences in **cybersecurity, data analytics, and web development**. Built using **Massively by HTML5 UP**, this platform is designed to provide an engaging and visually appealing experience.

## 🌟 Features

✅ **Project Showcase** – View my latest work in cybersecurity, data analytics, and web development. 💻  
✅ **About Me** – Learn more about my background and expertise. 🎓  
✅ **Contact** – Easily get in touch with me for collaboration opportunities. 📩  
✅ **Responsive Design** – Fully optimized for mobile and desktop devices. 📱  
✅ **Interactive UI** – Smooth scrolling and engaging visuals for a seamless experience. 🎨  

## 🚀 Technologies Used

- **HTML, CSS, JavaScript** – Frontend development.  
- **React & Tailwind CSS** – For a sleek, modern design.  
- **MongoDB** – Backend database for project details and user interactions.  
- **Node.js & Express.js** – API and server-side logic.  
- **Massively by HTML5 UP** – A beautifully designed template serving as the foundation.
